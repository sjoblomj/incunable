Miscellaneous

Commonmark features are supported, such as inlining code and commands like this: grep -Irin searchstring.

The GitHub flavour of Commonmark is used, meaning it has support for extensions like tables and Strikethrough.1

The engine supports templating common phrases. For example, there is a template to reference GitHub projects: The code is available at my GitHub account.2
